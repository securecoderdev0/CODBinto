##  Quick Setup (3 Steps)

### 1. Install Rainmeter (The Engine)
Rainmeter is the free tool that runs all the desktop widgets.
*   **Download:** [Rainmeter Official Site](https://www.rainmeter.net/)
*   **Install:** Run the setup and follow the standard installation prompts.
*   **Apply CODBinto:** Double-click your `CODBinto.rmsix` file. 
*   > [!IMPORTANT]
    > **CHECK THE BOX** that says "Load included skins" in the installer window!

### 2. Set Up Animated Backgrounds
Use Lively Wallpaper to achieve the "Ghost" look from the demo.
*   **Get the App:** Download [Lively Wallpaper](https://apps.microsoft.com/detail/9ntm2qc6qws7) from the Microsoft Store.
*   **Add Video:** Open Lively, click the **+ (Plus)** icon, and drag your `Ghost_Visual.mp4` file into the app.
*   **Apply:** Select your new video wallpaper to active the background.

### 3. Adjust to Your Screen
Every monitor is different. You can move the HUD elements to fit your layout.
*   **Move Elements:** Hold the `Ctrl` key and click-and-drag any widget to move it.
*   **Edit Values:** Right-click any widget and select **Edit Skin** to change colors or precise `X/Y` positions.
*   **Refresh:** Save your changes in the text file, right-click the widget, and select **Refresh Skin**.

##  Troubleshooting
*   **Skins aren't loading?** Right-click the Rainmeter tray icon, select "Manage", and make sure the `CODBinto` folders are active.
*   **PC is slow?** Go to Lively Wallpaper settings and set "Wallpaper Pause" to **"All Apps"** or **"Full Screen"**.
*   **Elements are overlapping?** Use `Ctrl + Drag` to reposition them or edit the UI coordinates in the `.ini` files.
